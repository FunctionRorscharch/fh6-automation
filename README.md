# FH6Auto v4.1

测试这东西是很麻烦的，电脑和电脑之间的差距比我和吴彦祖的差距都大，所以遇到问题多反馈多贴日志和图片。

FH6Auto 是一个基于 Python、图像识别和输入自动化的 FH6 本地辅助工具，当前主要支持：

- 循环跑图
- 批量买车
- 专精获取抽奖
- AI 辅助选车
- 诊断记录
- 自适应屏幕尺寸校准
- 根据CR调节买车数量

> 本项目仅用于自动化、图像识别和本地脚本技术交流学习。请勿用于商业用途或破坏游戏平衡。使用风险请自行承担。

## 项目来源

本项目基于 [YOUSTHEONE/FH6Auto](https://github.com/YOUSTHEONE/FH6Auto) 持续维护，并结合当前游戏版本与实际用户环境做了适配、重构和稳定性优化。

## 运行环境建议

- Windows
- 游戏语言：简体中文
- 输入法：英文键盘
- 建议分辨率环境：1080P
- 建议关闭 HDR
- 运行时保持游戏在前台

## 主要功能

### 循环跑图

- 自动进入 EventLab
- 自动输入蓝图代码
- 自动寻找赛事入口并重复执行

### 批量买车

- 自动进入车辆收藏与目标车辆页面
- 自动重复购买
- 支持 CR 总额度限制

### 超级抽奖

- 自动进入车辆菜单、选车、上车、点技能
- 支持 AI 辅助选车
- 支持智能页码

## 常用开关

- `AI辅助`：启用 AI 参与选车
- `智能页码`：启用页码记忆跳转
- `纯AI`：只使用 AI 识别，不回退模板
- `自动截图`：保存 AI 识别样本
- `诊断记录`：输出完整诊断报告

## 快捷键

- `F8`：停止任务
- `F3`：测试找图流程

## 从源码运行

标准环境：

```powershell
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install -r requirements.txt
python main.py
```

AI 环境：

```powershell
python -m pip install -r requirements-ai.txt
```

模型文件默认路径：

```text
models/fh6_car_select_yolo.pt
```
